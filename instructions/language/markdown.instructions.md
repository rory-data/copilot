# Markdown Standards

- **Language**: Use New Zealand English for spelling and grammar.
- **H1 usage**: Do not use H1 in content documents — H1 is reserved for document titles and instruction file headings.
- **Code blocks**: Use fenced code blocks for all code snippets. Always specify the language after the opening backticks for syntax highlighting.
- **Links**: Use descriptive link text — avoid "click here" or bare URLs. Ensure links are valid.
